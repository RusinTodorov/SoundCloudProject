# Soundcloud.com

## Final Project for IT Talets Season 12

### Building a React version of soundcloud.com with some basic fuctionality

# Features:

* Register/ login / logout 
* Uploading track
* Like & unlike track
* Search and filter tracks and users
* Play tracks
* Notifications

#  Technologies:

 * Javascript XML
 * React
 * Redux
 * HTML
 * CSS & SCSS modules
 * Material UI
 * Firebase
